## 📌 Project Overview

This project was completed as **Task 4** during my **Data Analytics Internship at Skill Craft Technology**.

The goal of this task is to perform **Exploratory Data Analysis (EDA)** on a customer marketing dataset to uncover patterns, trends, and actionable business insights through data visualization.

---

## 🎯 Objectives

✅ Understand dataset structure  
✅ Clean and preprocess data  
✅ Perform Exploratory Data Analysis  
✅ Create visual dashboards & charts  
✅ Extract business insights from data
